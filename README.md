# AIと現場と創作の学習記録

CodexやPython、Gitを少しずつ触りながら、AI創作や現場仕事での使い方も試していく個人ホームページです。
学ぶ途中で覚えたことやつまずいたことを、あとから振り返れるように残していきます。

## ファイル構成

- `index.html`: 日本語トップページ
- `learning.html`: 学習記録ページ
- `style.css`: レイアウトとレスポンシブデザイン
- `script.js`: スマホ用メニューの開閉
- `README.md`: この説明ファイル

## 確認方法

`index.html` をブラウザで開いてください。

外部ライブラリ、npm、React、Next.js、WordPressは使用していません。
