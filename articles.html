ホームページ第1版を作りました

Codexを使って、HTML / CSS / JavaScriptだけの小さなホームページを作りました。 トップページ、学習記録、記事メモ、このサイトについて、の4ページ構成まで形にしました。

Gitで変更履歴を残しながら進めましたが、PowerShellの手入力は思ったより負担が大きいと感じました。 そこで、git statusやgit diff、commit、表示確認はCodexに任せて、自分は監督する形に切り替えました。

AGENTS.mdも作り、今後の作業ルールをプロジェクト内に残しました。 まず小さく作って、壊さず育てる方針で進めていきます。
