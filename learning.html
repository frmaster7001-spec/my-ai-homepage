First Note

最初の学習メモ

01

小さなホームページ第1版を作った

Codexを使って、HTML / CSS / JavaScriptだけの小さなホームページ第1版を作りました。 Gitで変更履歴を残しながら進める中で、PowerShellの手入力が作業時間の負担になることにも気づきました。

そのため、git status、git diff、commit、表示確認はCodexに任せ、自分は監督する形に切り替えました。 AGENTS.mdも作成し、今後の作業ルールをプロジェクト内に残しました。

まず小さく作って、壊さず育てる方針で進めていきます。

Second Note

2本目の学習メモ

02

GitHub Pagesでホームページを公開し、画像と記事導線を整えた

GitHub Pagesを使って、HTML / CSS / JavaScriptだけで作った静的ホームページを公開しました。 トップページにはアイキャッチ画像を追加し、公開URLから表示できることも確認しました。

記事一覧にはサムネイル用の枠を作り、1本目の記事には実際のサムネイル画像を追加しました。 さらに、個別記事ページの下部に「記事一覧へ戻る」「トップページへ戻る」導線をそろえて、読み終わったあとに移動しやすくしました。

READMEと実ファイル構成のズレも修正しながら、Codexには変更予定を提示させ、承認後に編集・commit・pushする流れを維持しました。 小さく進めつつ、公開後も壊さず育てる練習になっています。

Third Note

3本目の学習メモ

03

公開URLの扱いを見直し、学習用の実験室として運用する方針にした

GitHub Pagesでホームページを公開できる状態まで進めました。 ただ、公開URLに外部発信にそのまま使うには少し気になる文字列が含まれるため、Xなどに直接載せるのは避ける判断をしました。

今後は、このホームページを学習用の実験室・実験記録として使います。 外部発信では、URLではなく、学習内容、気づき、スクリーンショット、考察を安全な範囲で出していく方針です。

将来、別名義URL、別GitHubアカウント、独自ドメインなど安全な導線が整った段階で、正式な発信拠点として再公開します。 危ない可能性があるものは手元に置き、危なくないものだけ外に出すことにしました。
